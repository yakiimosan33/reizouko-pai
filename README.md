# StockPi Lite LP

React + Vite + Tailwind CSS のLPです。

## Local

```bash
npm ci
npm run dev
```

## Build

```bash
npm run build
```

出力先は `dist/` です。

## Vercel

Vercel に GitHub リポジトリを連携して、そのままデプロイできます。

- Framework Preset: `Vite`
- Install Command: `npm ci`
- Build Command: `npm run build`
- Output Directory: `dist`
- Node.js Version: `22.x`
